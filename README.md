# Smart Logistics Management & Analytics Platform

## Overview
A MySQL-backed Streamlit web application for logistics tracking and analytics.

## Tech Stack
- Python
- MySQL
- Streamlit

## Features
- Real-time shipment tracking
- Operational KPI monitoring
- Delivery performance insights
- Courier performance benchmarking
- Cost analysis
- Cancellation analysis
- Warehouse insights

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run myapp.py

## Database Setup
Import the provided CSV and JSON files into MySQL before running the application.
Update database credentials in db_connection.py.
